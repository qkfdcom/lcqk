# 链查区块 (LCQK)

## 项目简介
链查区块（LCQK）是一个专业的 Web3 用户信息查询系统，提供币圈用户的备注信息查询、分类管理等功能。

## 功能特性
- 查询功能
  - 支持用户ID查询
  - 支持关键词模糊搜索
  - 实时搜索结果展示

- 分类展示
  - 普通名单：一般用户信息
  - 黄名单：需要警惕的用户
  - 黑名单：高风险用户

- 数据管理
  - 本地JSON文件存储
  - 支持数据更新
  - 分类管理

## 技术栈
- 前端框架：Next.js 14
- UI框架：Tailwind CSS
- 开发语言：TypeScript
- 代码规范：ESLint
- 部署平台：Vercel
